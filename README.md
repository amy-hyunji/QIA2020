# QIA2020
code for QIA2020
https://www.kaggle.com/c/qia-hackathon2020

## Goal
- multimodal sentiment analysis model
- 7 labels (hap, ang, dis, fea, neu, sad, sur)
- dataset (train: 44370, val: 5386)
    - video
    - text embedding

## How to train
```python train.py```


## How to test
```python test.py --name test```
- this will save test.csv under the `data-dir` you assigned
